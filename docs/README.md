# Documentation Archive

This folder holds non-production material retained for future maintenance. It is not required to serve the website.

- `analytics/`: analytics evidence
- `deployment/`: host-specific fallback notes and configuration
- `integrations/`: Google Sheets discovery-call setup material
- `migration/`: blog and legacy SEO migration records
- `strategy/`: approved copy, prompts, positioning, and visual notes
- `tools/`: archived QA and migration utilities
